1. API 계층 (`/app/api/v1/endpoints/ocr.py`)
       * 사용자가 /upload 경로로 이미지를 업로드하면, 이미지 파일을 읽어 OCR 서비스로 전달합니다.

   2. 서비스 계층 (`/app/services/ocr_service.py`)
       * 전달받은 이미지를 pytesseract 라이브러리를 사용해 처리합니다.
       * 이때 한글(`kor`)과 영어(`eng`)를 모두 인식하여 이미지에서 텍스트를 추출합니다.
       * 추출된 텍스트에서 불필요한 줄바꿈이나 공백을 제거한 후, 깨끗한 텍스트를 다시 API 계층으로 돌려줍니다.

   3. 데이터베이스 계층 (`/app/crud/crud_product.py`)
       * API 계층이 서비스로부터 받은 텍스트를 이 곳으로 넘겨 DB 검색을 요청합니다.
       * 여기서 최종적으로 DB를 조회합니다.




   1. OCR로 추출된 텍스트를 공백 기준으로 단어들로 분리합니다. (예: "눈 건강 루테인" -> ['눈', '건강', '루테인'])
   2. Product (제품) 테이블에서 이 단어들이 포함된 제품을 찾습니다.
   3. 검색 대상이 되는 컬럼은 두 가지입니다.
       * PRDLST_NM: 제품명
       * PRIMARY_FNCLTY: 제품의 주된 기능성
   4. OR 조건으로 검색하므로, 추출된 단어 중 하나라도 제품명 또는 주된 기능성에 포함되면 검색 결과에 포함시킵니다.
   5. 일치하는 제품 중 상위 3개만 가져옵니다.
